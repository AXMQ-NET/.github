# AXMQ | ApexMQTT (鼎极)

> **在毫秒被定义之前，万物已然抵达。**
> **Arrival, Before the Millisecond Begins.**

AXMQ (ApexMQTT) 是一款面向百万级设备接入、千万级消息分发而生的极致高性能 MQTT 代理（Broker）。

### 🌌 核心哲学：时空重构
在 AXMQ 的世界里，速度不再是追逐，而是一种“抵达”。通过对底层网络栈的深度重构，我们消灭了每一毫秒的冗余。

- **零拷贝 (Zero-Copy):** 数据流转如影随形，无需搬运。
- **多核异步 I/O:** 彻底解放 CPU，压榨每一丝吞吐潜力。
- **鼎极性能:** 它是单机连接的物理极境。
